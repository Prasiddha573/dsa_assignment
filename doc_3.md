# Program 3: Reverse Traversal of Linked List

## Data Structure Used
A singly linked list is used.  
Each node contains an integer data and a pointer to the next node.

## Functions Implemented
reversePrint() displays the linked list elements in reverse order using recursion.

## Organization of main()
The main function creates a linked list by taking input values.  
It then calls the reversePrint function to print the list in reverse order.

## Sample Output

Input:
5
10 20 30 40 50

Output:
50 40 30 20 10
