# Program 8: Sorting Random Numbers

## Data Structure Used
An array is used to store random integers.  
Another array is used to preserve original data.

## Functions Implemented
bubbleSort() sorts using bubble sort.  
selectionSort() sorts using selection sort.  
insertionSort() sorts using insertion sort.  
mergeSort() sorts using merge sort.  
merge() merges two sorted subarrays.

## Organization of main()
The main function reads the value of N.  
It generates random numbers between 1 and 1000.  
It asks the user to select a sorting algorithm.  
It prints the array before sorting.  
It sorts the array using the chosen method.  
It prints the sorted array and operation counts.

1
