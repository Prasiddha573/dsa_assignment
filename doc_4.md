# Program 4: Doubly Linked List Implementation

## Data Structure Used
A doubly linked list is implemented using a structure.  
Each node contains data, a pointer to the previous node, and a pointer to the next node.

## Functions Implemented
insertAfter() inserts a new node after a given node value.  
deleteNode() deletes a node with the given value.  
display() prints the elements of the list.

## Organization of main()
The main function creates a doubly linked list using input values.  
It demonstrates insertion after a node and deletion of a node.  
Finally, it displays the updated list.

## Sample Output

Input:
4
10 20 30 40

Output:
10 20 30 40
